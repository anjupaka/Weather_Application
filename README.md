# 🌦️ Weather Dashboard App

A beautifully designed and responsive **Weather Dashboard Application** built using **React.js**, **Vite**, and **Material UI**. This app provides real-time weather updates, air quality index, a 5-day forecast, and sunrise/sunset information – all presented in a modern, clean UI.

🔗 **Live Demo**: [myweatherapp-dev.netlify.app](https://myweatherapp-dev.netlify.app)

---

## 🚀 Features

- 🔍 Search weather by **City Name**
- 📍 Get **Current Location** weather data
- 🌡️ Real-time **Current Weather** info
- 🌬️ **Air Quality Index** display
- 📅 **5-Day Forecast** with temperature, icons & description
- 🌅 **Sunrise & Sunset** timings
- 📱 Fully **Responsive Design**
- 🎨 Modern UI with **gradient blue-purple theme**

| Tech | Description |
